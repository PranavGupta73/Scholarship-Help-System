For this program we require Apache NetBeans IDE.

1. Run the given sql file to create database.
2. Using an IDE (preferably NetBeans) open the code folder.
3. In the following lines change "pranavgupta" (which is my password for mysql workbench) with your password :-
	DB.java line 13
	login.java line 156
	scholarshipDisplay.java line 33
	viewScholarships.java line 30
4. After changing the password at all of above locations first build the project and then run the program using "run (green arrow)" in NetBeans.
 
The program will run as shown in the video if all steps are followed.
